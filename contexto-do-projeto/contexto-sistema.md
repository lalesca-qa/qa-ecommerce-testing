# Contexto do Sistema

Este projeto simula testes em um e-commerce real de treinamento:

https://www.saucedemo.com

## Objetivo do sistema
Permitir que usuários realizem login, naveguem por produtos, adicionem itens ao carrinho e finalizem compras.

## Perfil do usuário
- Usuário final padrão
- Sem níveis de permissão complexos

## Risco do sistema
Fluxo de compra é crítico (alta prioridade de testes)
